# 分别解释 Ridge 回归和 Lasson 回归

## 回答重点

Ridge 回归和 Lasso 回归是两种常用于**处理多重共线性**和**防止过拟合**的正则化技术，它们都在最小化线性回归模型的残差平方和的目标函数中加入了一个正则化项。

1）**Ridge 回归（岭回归）**：也叫 Tikhonov 正则化。它在目标函数中添加了所有回归系数的平方和作为罚项。公式如下：
$$
\text { 目标函数 }=\min \left(\sum_{i=1}^n\left(y_i-\beta_0-\sum_{j=1}^p \beta_j x_{i j}\right)^2+\lambda \sum_{j=1}^p \beta_j^2\right)
$$
其中，$λ$ 是正则化参数，控制着罚项的强度，防止系数过大，从而减小模型复杂度。

2）**Lasso 回归（最小绝对收缩和选择算子）**：它在目标函数中添加了所有回归系数的绝对值和作为罚项。公式如下：
$$
\text { 目标函数 }=\min \left(\sum_{i=1}^n\left(y_i-\beta_0-\sum_{j=1}^p \beta_j x_{i j}\right)^2+\lambda \sum_{j=1}^p |\beta_j| \right)
$$
Lasso 回归相比 Ridge 回归有一个显著的特点就是它可以产生稀疏模型，即部分回归系数可以被压缩到零，这样便于特征选择。

## 扩展知识

1）**特征选择**

- Ridge 回归不会产生稀疏解，它会缩小所有系数但不会将其置零。因此，它适用于特征很多但都可能有一定影响的情况。
- Lasso 回归由于 L1 惩罚会使得一些系数变为零，这实际上相当于进行了特征选择，非常适合处理高维数据。

2）**正则化参数的选择**

- 两者共同点是都需要选择合适的正则化参数 $λ$，这通常通过交叉验证来确定。
- 需要注意的是，过大的 $λ$ 值会导致模型过于简单，而过小的 $λ$ 值则可能无法有效避免过拟合。

3）**组合使用：弹性网（Elastic Net）**

- 在实际应用中，有时会混合使用 Ridge 和 Lasso 的正则化技巧，这就是所谓的弹性网（Elastic Net）。其正则化项是 L1 和 L2 的结合，能够同时拥有两者的优点，公式如下：

$$
\text { 目标函数 }=\min \left(\sum_{i=1}^n\left(y_i-\beta_0-\sum_{j=1}^p \beta_j x_{i j}\right)^2+\lambda_1 \sum_{j=1}^p\left|\beta_j\right|+\lambda_2 \sum_{j=1}^p \beta_j^2\right)
$$

弹性网将会针对不同的数据集给予更灵活的模型调优。

实际工程和论文里，弹性网一般也写成：
$$
\min \sum (y_i - \hat y_i)^2 + \lambda \left( \alpha \sum|\beta_j| + (1-\alpha)\sum\beta_j^2 \right)
$$
4）**计算效率和模型解释性**

- Ridge 回归计算效率较高，并且模型解释性强，因为所有特征都会保留下来。
- Lasso 回归则由于其稀疏性，在解释模型的时候会更加直观。然而，计算效率在某些高维度情况下会略差。

5）**实际应用例子**

- 在财务数据分析中，Ridge 回归经常被用来处理众多相互之间有相关性的财务指标。
- 在文本分类或者基因数据分析中，Lasso 回归的特征选择功能特别有用，可以帮助识别最关键的词语或者基因。
